# Contributing to Melitta Barista Card

Thanks for your interest in contributing!

## Development Setup

```bash
git clone https://github.com/dzerik/melitta-barista-card.git
cd melitta-barista-card
npm install
npm run build
```

## Development

```bash
npm run dev   # watch mode with auto-rebuild
```

Copy `dist/melitta-barista-card.js` to your HA `config/www/` directory for testing.

## Pull Request Process

1. Fork the repository and create a branch.
2. Make your changes.
3. Ensure `npm run build` succeeds without errors.
4. Submit a PR with a clear description.
